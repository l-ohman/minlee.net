# minlee.net

## Editing site data

| Content             | File location                   |
| ------------------- | ------------------------------- |
| Biography           | `/src/components/Bio/index.tsx` |
| Selected Highlights | `/src/data/highlights.tsx`      |
| News                | `/src/data/news.tsx`            |

## Project setup

This project requires Node 24 and yarn.

Install packages:

```
yarn
```

Start dev server:

```
yarn dev
```
